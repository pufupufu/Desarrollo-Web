Párrafos

<p>...</p>
El atributo "align" puede tener 4 valores: "center", "left", "right", "justify".

<pre>...</pre> altera el comportamiento del navegador para que mantenga los espacios o saltos de linea de un parrafo tal cual estan en el codigo HTML.

Lineas horizontales

<hr> crea una linea horizontal
Atributos:
  • align
  • noshade
  • size
  • width

Etiquetas de estilo físico

  • Texto en negrita: <b>...</b>
  • Texto en cursiva: <i>...</i>
  • Texto monoespaciado: <tt>...</tt>
  • Texto tachado: <strike>...</strike>
  • Aumentar tamaño del texto: <big>...</big>
  • Reducir tamaño del texto: <small>...</small>
  • Poner subindices: <sub>...</sub>
  • Poner superindices: <sup>...</sup>
  • Texto subrayado: <u>...</u>

Etiquetas de estilo lógico

  • Indica que el texto marcado es de mayor importancia: <strong>...</strong>
  • Muestra el texto en cursiva: <em>...</em>
  • Texto que forma parte de un codigo: <code>...</code>
  • Produce el mismo efecto que <code>...</code>: <samp>...</samp>
  • Delimita texto escrito por el usuario: <var>...</var>
  • Se usa para citas o referencias: <cite>...</cite>
  • Se usa para introducir una definicion: <dfn>...</dfn>

Listas

  • Listas no ordenadas
    • Delimitadas por: <ul>...</ul>
    • Cada elemento por: <li>...</li>
  • Listas ordenada o numeradas
    • Delimitadas por: <OL>...</OL>
    • Cada elemento por: <li>...</li>
  • Listas de glosario
    • Delimitadas por: <DL>...</DL>
    • Para la palabra se usa: <DT>
    • Para la descripcion de la palabra se usa: <DD>
Como en el listado anterior, las listas pueden ser anidadas.

Comentarios
