Con la aparicion de los valores "right" y "left", también se introdujo un nuevo atributo a la etiqueta <br/>: "clear"

  • <br clear="left"/> Provoca un salto de linea hasta el primer margen libre a la izquierda
  • <br clear="right"/> Provoca un salto de linea hasta el primer margen libre a la derecha
  • <br clear="all"/> Provoca un salto hasta la primera linea que tiene ambos margenes libres

También hay dos nuevos atributos para la etiqueta <img>: "vspace" y "hspace" que controlan el tamaño de los margenes vertical y horizontal respectivamente.

El tamaño de los márgenes se indica en píxeles.

Por ejemplo: <img align="left" vspace="35" hspace="35">

Es posible modificar el tamaño de las imágenes usando los atributos "width" y "height" para ancho y alto respectivamente.

También existe el atributo "alt" que permite especificar un texto alternativo en caso de que la imagen no pueda ser vista.

Las imágenes externas no se visualizan con el resto de la página, sino que se proporciona un enlace para que se pueda acceder a ellas.

Por ejemplo: <a href="imagen.xxx">imagen externa</a>

Etiquetas de estilo físico

  • Texto en negrita: <b>...</b>
  • Texto en cursiva: <i>...</i>
  • Texto monoespaciado: <tt>...</tt>
  • Texto tachado: <strike>...</strike>
  • Aumentar tamaño del texto: <big>...</big>
  • Reducir tamaño del texto: <small>...</small>
  • Poner subindices: <sub>...</sub>
  • Poner superindices: <sup>...</sup>
  • Texto subrayado: <u>...</u>

Etiquetas de estilo lógico

  • Indica que el texto marcado es de mayor importancia: <strong>...</strong>
  • Muestra el texto en cursiva: <em>...</em>
  • Texto que forma parte de un codigo: <code>...</code>
  • Produce el mismo efecto que <code>...</code>: <samp>...</samp>
  • Delimita texto escrito por el usuario: <var>...</var>
  • Se usa para citas o referencias: <cite>...</cite>
  • Se usa para introducir una definicion: <dfn>...</dfn>

Listas

  • Listas no ordenadas
    • Delimitadas por: <ul>...</ul>
    • Cada elemento por: <li>...</li>
  • Listas ordenada o numeradas
    • Delimitadas por: <OL>...</OL>
    • Cada elemento por: <li>...</li>
  • Listas de glosario
    • Delimitadas por: <DL>...</DL>
    • Para la palabra se usa: <DT>
    • Para la descripcion de la palabra se usa: <DD>
Como en el listado anterior, las listas pueden ser anidadas.

Enlaces hipertexto

Son zonas especiales de nuestro texto que ofrecen la posibilidad de pulsar sobre ellas para ir a otras paginas.

  • Enlaces internos o locales: enlazan las paginas que componen un mismo sitio web.
  • Enlaces externos: permiten ir a paginas en otro servidor web.

Para crear enlaces se usa la etiqueta <a>...</a>
El atributo "href" permite indicar cual es la pagina a la que debe saltar cuando el usuario lo presiona.
La estructura de un link es: <a href="direccionURL">Texto que será sensible (hipertexto)</a>

  • Enlace interno: <a href="varios/hobbies.html">Hobbies</a>
  • Enlace externo: <a href="https://www.youtube.com">Youtube</a>
Creación de anclas:

<a name="nombreAncla">Comienzo</a>
<a href="#nombreAncla">A Comienzo</a>

El atributo "target" indica dónde tiene que abrirse el enlace.

  • _blank: abrir el enlace en una nueva ventana
  • _self: abrir el enlace en la misma ventana donde está el código actual
  • _parent: abre el enlace en el frame padre
  • _top: abre la ventana en toda la ventana

También se pueden usar imágenes como enlace:
En primer lugar se debe insertar la instrucción de inicio de <a>, después incluir la etiqueta <img> y por último insertar </a>

Color del texto y links

Atributos de la etiqueta <body>:
  • text: controla el color que tendrá el texto
  • link: controla el color que tendrán los links
  • vlink: controla el color de los links visitados
  • alink: controla el color del enlaca activo

Imágenes de fondo

Para insertar una imágen de fondo se utiliza el atributo "background" de la etiqueta <body>.
Por ejemplo: <body background="fondo1.jpg">
Donde "fondo1.jpg" es la imagen elegida como fondo y se encuentra en el mismo directorio que la página.
